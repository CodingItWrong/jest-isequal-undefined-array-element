# jest-isequal-undefined-array-element

Reproduces facebook/jest issue 13885: isEqual ignores trailing undefineds in arrays

## Requirements

- [Node](https://nodejs.org)
- [Yarn 1.x](https://classic.yarnpkg.com/lang/en/)

## Installation

- Clone the repo
- Run `yarn install`

## Unit Tests

- Run `yarn test`
